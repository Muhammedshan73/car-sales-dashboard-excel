# Car Sales Dashboard | Excel Project

This project presents an interactive **Car Sales Dashboard** created in Microsoft Excel to analyze global automobile sales data. It visualizes total revenue, yearly trends, country-wise performance, car conditions, brand popularity, and customer preferences through PivotTables, PivotCharts, and slicers.
